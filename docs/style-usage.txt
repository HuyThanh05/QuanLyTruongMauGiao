HƯỚNG DẪN SỬ DỤNG style.css ĐỂ CHỈNH HTML

1) Màu thương hiệu (tím)
- Nằm trong :root của style.css: --bs-primary
- Đổi màu thương hiệu (toàn site): chỉnh giá trị --bs-primary và --bs-primary-rgb

2) Tiêu đề/breadcrumb/link thương hiệu
- Class: text-brand
- Cách dùng: <a class="text-brand">Trang chủ</a>
- Tác dụng: gán màu tím thương hiệu, hover đậm hơn một bậc

3) Nút tím thương hiệu
- Class: btn-brand
- Cách dùng: <button class="btn btn-brand">...</button>
- Tác dụng: nền tím thương hiệu, chữ trắng, hover đậm hơn
- Gợi ý: kết hợp với các class Bootstrap khác như btn-sm, rounded-0, px-3

4) Nút tròn nhỏ (Floating Action Button mini)
- Class: btn-fab
- Cách dùng: <button class="btn btn-brand btn-fab rounded-circle">...</button>
- Tác dụng: cố định kích thước 38x38, căn giữa icon

5) Header/NavBar
- Giảm chiều cao header: chỉnh padding trong selector .navbar { padding-top/bottom }
- Kích thước chữ brand: .navbar .navbar-brand { font-size }
- Kích thước nút trong navbar: .navbar .btn { padding, font-size }

6) Sidebar tím nhẹ
- Khối sidebar dùng class: sidebar sidebar-purple
- Các biến có thể chỉnh để đổi sắc độ:
  --sidebar-bg: nền sidebar (tím rất nhạt)
  --sidebar-border: viền phải
  --sidebar-text: màu chữ/icon
  --sidebar-hover: nền khi hover
- Link mục: .sidebar-purple .nav-link
  - Góc: border-radius (mặc định vuông 0)
  - Khoảng cách: padding (0.5rem 0.75rem)
- Thêm icon Font Awesome: <i class="fa-solid fa-house me-2"></i>

7) Footer
- Khối: .app-footer
- Chỉnh nền/viền/hover link tại các selector .app-footer, .app-footer .nav-link, .app-footer .nav-link:hover

8) Bảng, phân trang, form
- Sử dụng class Bootstrap mặc định: table, table-light, pagination, input-group
- Icon tìm kiếm: <span class="input-group-text"><i class="fa-solid fa-magnifying-glass"></i></span>

9) Quy ước khi thêm trang mới
- Mọi trang đều extends base.html và đặt nội dung trong block content
- Style mới nên đưa vào style.css thay vì inline nếu cần tái sử dụng
- Tạo class mới theo tiền tố hợp lý, tránh ghi đè Bootstrap không cần thiết

10) Gợi ý tuỳ biến nhanh
- Đổi toàn bộ sắc tím: thay --bs-primary trong :root
- Làm sidebar đậm/nhạt hơn: chỉnh --sidebar-bg và --sidebar-text
- Làm header thấp/thoáng hơn: chỉnh padding của .navbar

11) Ví dụ nhanh
- Link tím: <a class="text-brand" href="#">Xem chi tiết</a>
- Nút hành động chính: <button class="btn btn-brand btn-sm">Lưu</button>
- Nút thêm nhỏ tròn: <button class="btn btn-brand btn-fab rounded-circle"><i class="fa-solid fa-plus"></i></button>
