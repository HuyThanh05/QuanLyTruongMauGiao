LOGINFORM - HƯỚNG DẪN SỬ DỤNG CSS VÀ JS

1) Vị trí file liên quan
- HTML: app/templates/login.html
- CSS:  app/static/loginform.css
- (Tuỳ chọn) JS: có thể tách ra app/static/login.js rồi include vào template

2) Bố cục chính (HTML classes)
- .container: Khung tổng, bao quanh 2 form và khu vực toggle
- .form-box.login: Form Đăng nhập (hiển thị mặc định)
- .form-box.register: Form Đăng ký (hiển thị khi container có class "active")
- .toggle-box: Nền hiệu ứng phía sau panel
- .toggle-panel.toggle-left: Panel bên trái (tiêu đề + text + nút Register)
- .toggle-panel.toggle-right: Panel bên phải (tiêu đề + text + nút Login)
- .input-box input: Ô nhập liệu Username/Password/Email/...
- .forgot-link a: Liên kết "Forgot password?"
- .social-icons a: Nút MXH (Facebook/Google/Twitter)
- .btn: Nút chung; riêng nút Login dùng selector .form-box .btn, nút Register dùng .toggle-panel .btn

3) Màu sắc chủ đạo (theme tím)
- Tím chính: #6f42c1, #8850ee
- Tím gradient/nút: #a97ff7, #b992f4
- Màu phụ hover: #5a35b0, #532bb0
- Nền nhẹ: #f3ecff, #eadbff, #f5f2fa, #ece7fc

4) Nền trang (background)
- Mặc định dùng gradient tím nhạt:
  body {
    background: linear-gradient(135deg, #ffffff 0%, #f3ecff 35%, #eadbff 100%);
  }
- Tuỳ chỉnh nhanh:
  - Nhạt hơn: thay #eadbff -> #faf7ff
  - Rõ hơn: thay #eadbff -> #efe6ff hoặc giữ #eadbff nhưng tăng tỉ lệ phần trăm điểm cuối

5) Ô nhập (input)
- Chiều cao/độ lớn: chỉnh tại .input-box input
  + padding: tăng giảm để ô "dài" hơn (ví dụ 16px 54px 16px 22px)
  + font-size: 17px (có thể 16–18 tuỳ ý)
  + màu nền ô: background: #eee (có thể đổi)

6) Liên kết quên mật khẩu
- Căn trái: .forgot-link { text-align: left; }
- Màu tím đồng bộ: .forgot-link a { color: #8850ee; }
- (Gợi ý) Hover: .forgot-link a:hover { color: #5a35b0; }

7) Nút Login (trong form)
- Nút mặc định: .btn { background: #a97ff7; color: #fff; height: 50px; border-radius: 14px; }
- Hover/Focus đồng bộ theme:
  .form-box .btn:hover,
  .form-box .btn:focus {
    background: linear-gradient(90deg, #a97ff7 0%, #8850ee 100%);
    color: #ffffff;
    box-shadow: 0 6px 18px rgba(111, 66, 193, 0.18);
  }
- Tuỳ chỉnh nhanh:
  - Đậm hơn: tăng độ tương phản gradient (#7f3fe6)
  - Dịu hơn: dùng nền #ece7fc thay gradient, giữ chữ #532bb0 hoặc #5a35b0

8) Nút Register (ở panel)
- Nút mặc định:
  .toggle-panel .btn {
    width: 160px; height: 46px;
    background: linear-gradient(to left, #ffffff 70%, #b992f4);
    border: 2px solid #a087d8;
    color: #7046b7; border-radius: 15px;
  }
- Hover/Focus đồng bộ theme:
  .toggle-panel .btn:hover,
  .toggle-panel .btn:focus {
    background: linear-gradient(to left, #f3ecff 60%, #a97ff7);
    color: #5a35b0; border-color: #a97ff7;
    box-shadow: 0 4px 16px rgba(111, 66, 193, 0.14);
  }
- Khoảng cách với dòng text phía trên: điều khiển bởi gap ở .toggle-panel hoặc riêng .toggle-panel.toggle-left { gap: 6px; }

9) Social icons (MXH)
- Nền có gradient mờ, bo góc 20px, kích thước 50x50
- Chỉnh màu trong .social-icons a (linear-gradient(...))

10) Hiệu ứng panel chuyển đổi (toggle-box)
- Hiệu ứng dải gradient chạy ngang được tạo bởi ::before của .toggle-box
- Có thể thay đổi màu/tốc độ bằng chỉnh gradient và vị trí trong .toggle-box::before

11) Gợi ý chỉnh nhanh theo nhu cầu
- Muốn nền trang tím hơn: tăng tỉ lệ/tông màu điểm cuối gradient của body
- Muốn ô nhập dài hơn: tăng padding dọc (ví dụ 18–20px) hoặc đặt height: 52px
- Muốn nút sát chữ hơn: giảm gap của .toggle-panel.toggle-left xuống 4–6px
- Muốn liên kết nổi bật hơn: thêm underline hoặc đậm màu hover

12) JavaScript chuyển đổi giữa Login/Register
- Mục tiêu: nhấn "Register" mở form Đăng ký, nhấn "Login" về form Đăng nhập
- Cơ chế: thêm/bỏ class "active" vào .container

JS mẫu (có thể đặt trong app/static/login.js):

// login.js - Toggle giữa Login/Register
(function () {
  const container = document.querySelector('.container');
  const toRegisterBtn = document.querySelector('.register-btn');
  const toLoginBtn = document.querySelector('.login-btn');

  if (!container) return;

  if (toRegisterBtn) {
    toRegisterBtn.addEventListener('click', function (e) {
      e.preventDefault();
      container.classList.add('active');
    });
  }

  if (toLoginBtn) {
    toLoginBtn.addEventListener('click', function (e) {
      e.preventDefault();
      container.classList.remove('active');
    });
  }
})();

Cách nhúng JS vào template (login.html):
- Thêm trước </body>:
  <script src="{{ url_for('static', filename='login.js') }}"></script>

13) Quy ước và giữ đồng bộ
- Giữ tông tím chủ đạo: #6f42c1, #8850ee, #a97ff7, #5a35b0
- Khi đổi màu, đồng bộ cả: nút, link, hover, gradient nền
- Tránh dùng quá nhiều tông đậm cùng lúc để không chói mắt; ưu tiên tím nhạt ở nền, tím rõ ở điểm nhấn

Kết thúc.
